${ProjName}: cap_CTIMER0

Progetto usato per sperimentare il capture 
con CTIMER0 e uso di interrupt

hardware necessario: LPC845-BRK e LCD

Sull'LCD viene presentato il valore di secondi che 
passano tra due pressioni successivi del 
pulsante USER

  
This example runs from Flash. 


To run this code: 1. Build
                  2. Load to flash
                  3. Reset
              or:
                  1. Build
                  2. Download and Debug
                  3. Run or Go


Note: It may be necessary to power-cycle the board in order to regain control of the reset button after programming.
